idiginBPEL

Generador de invariantes para WS-BPEL

idiginBPEL es un generador de invariantes para el lenguaje WS-BPEL. Actuando como interfaz gráfica para Takuan el sistema permite, a partir de un fichero BPEL, ejecutar casos de prueba, analizar el comportamiento del BPEL en los mismos y generar invariantes que permitan estudiarlo.
