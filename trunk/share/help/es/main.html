IdiginBPEL

Generador de invariantes para WS-BPEL

  • Sobre IdiginBPEL

    idiginBPEL es un generador de invariantes para el lenguaje WS-BPEL.
    Actuando como interfaz gráfica para Takuan el sistema permite, a partir de un fichero BPEL, ejecutar casos de prueba, analizar el comportamiento del BPEL en los mismos y generar invariantes que permitan estudiarlo.

  • Objetivos

    La razón del desarrollo de IdiginBPEL es permitir la ejecución masiva de test de forma sencilla así como facilitar una mejor manera de organizar el contenido generado por la aplicación (por ejemplo invariantes). La interfaz en línea de comandos original de Takuan requiere de la edición y el manejo directo de ficheros y solo permite al usuario la ejecución de pruebas en lote, siguiendo el proceso completo e incluso repitiendo etapas innecesarias. Esto ha sido solucionado aislando cada etapa del proceso, de manera que el usuario puede realizarlas individualmente, permitiendo un control más granulado.

  • Breve introducción

    Takuan sigue un proceso de tres pasos en la obtención de invariantes a partir de una composición BPEL y un conjunto de casos de prueba BPTS.

    1. Instrumentado del BPEL y resolución de dependencias
      Resuelve los import del BPEL y construye un paquete dispuesto para ser ejecutado en el servidor
    2. Ejecución de Tests
      Ejecuta el conjunto de casos de prueba BPTS contra el servidor, guardando los logs
    3. Analizar las trazas de ejecución
      4. Convierte los logs en el formato de trazas de Daikon y las analiza para obtener invariantes.

    Idiginbpel separa cada etapa, de manera que el usuario puede fácilmente repetir una o cambiarla sin necesidad de empezar de nuevo todo el proceso desde el principio. La herramienta además evita el manejo de ficheros para configurar las pruebas, para trabajar solo necesita el programa BPEL y el conjunto de casos de prueba BPTS.
