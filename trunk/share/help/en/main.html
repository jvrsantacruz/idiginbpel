IdiginBPEL

Automatic invariant generator for WS-BPEL

  • About IdiginBPEL

    IdiginBPEL is an automatic invariant generator for the WS-BPEL language.
    As a graphic interface for Takuan, the application allows to run BPTS test suites and analyze BPEL program behaviour by calculating invariants that can be used to improve the compositions and debugging.

  • Objetives

    The reason of IdiginBPEL is to make easier to run massive tests and provide a reasonable way to manage and organize generated content (i.e invariants). The Takuan's original command-line interface require the direct edition and management of files and only allows users to run batch tests, following the complete process, even repeating expensive and unnecesary steps. This has been solved by isolating every step of the workflow, so the user can use it individually, with a more fine-grained process control.

  • Brief introduction

    Takuan has a three step workflow to obtain invariants from a BPEL composition and a BPTS test suite.

    1. Instrument BPEL and resolve dependences
      Resolve BPEL imports and build a package ready to run on the server
    2. Run tests
      Run the BPTS test suite against the server storing the logs
    3. Analyze execution traces
      Convert logs into a Daikon trace format and analyze it to obtain invariants.

    IdiginBPEL separate each step so the user can easily repeat one or change it without have to start again and run the entire process from the begining. The tool also avoid file management to configure tests, only needs to work a BPEL program and the BPTS test suite.
